**Fly through a living cosmos. Collect essence. Evolve through 13 phases. Build, defend, battle, and ascend.**

A single-file HTML5 game that merges GTA-style open world, Sims life simulation, base building, tower defense, and arena combat into one interconnected evolution loop.

## ✦ Features

| Category | Details |
|----------|---------|
| **Open World** | 3,000 × 2,000 px world with 5 named zones, smooth camera scrolling, mini-map |
| **Life Sim** | Hunger, energy, social, fun needs — decay over time with penalties and structure bonuses |
| **Base Building** | 6 structure types (Essence Geyser, Meditation Chamber, Nexus Hub, Stellar Turret, Void Wall, Cosmic Lab) |
| **Tower Defense** | Wave-based enemies chase your character, turrets auto-attack, escalating difficulty |
| **Battle Arena** | Challenge any of 7 agents to 1v1 combat with attack/dodge mechanics |
| **Roaming NPCs** | 7 agent NPCs wander the world with proximity interaction dialogs |
| **Evolution** | 5 paths (Primordial, Plant, Quantum, Cosmic, Void) × 13 phases = 65 unique forms |
| **8 Abilities** | Click Power, Auto-Generation, Grow, Collapse, Absorb Star, Meditate, Harmonize, Devour |
| **Resources** | 6 core resources + building materials, 60+ resource nodes, random events |
| **Alignment** | Build reputation with 7 council agents through actions and arena victories |
| **Ascent Mode** | Start with an agent invocation for a custom challenge run |
| **Moon Phases** | 8 moon phases with dynamic bonuses that affect gameplay |
| **Konami Code** | ↑↑↓↓←→←→ — instant evolution unlock |

## 🎮 How to Play

| Action | Control |
|--------|---------|
| Move | `WASD` or `Arrow Keys` |
| Click to Move | Left-click on the world |
| Interact with NPCs | Click on them when close |
| Evolve | `E` key |
| Build Mode | `B` key |
| Open Abilities | Click ability buttons in HUD |
| Meditate | `M` key |
| Harmonize | `H` key |
| Absorb | `A` key |
| Launch | `L` key (phase 7+) |
| Transmit | `T` key (after launch) |
| Arena | Open alignment panel → click agent → Challenge |
| Close Panels | `Escape` |
| Konami Code | `↑↑↓↓←→←→` |

### Strategy Tips
- **Early game:** Fly through glowing orbs to collect essence. Evolve when you can.
- **Mid game:** Build structures to sustain your needs. Place turrets before waves hit.
- **Late game:** Challenge agents in the arena. Build Cosmic Labs for research bonuses.
- **Always:** Keep an eye on your needs bars — depleted needs drain your essence and HP.

## 🏗 Project Structure

```
Apophenoth/
├── index.html          # The entire game (single-file HTML + CSS + JS)
├── assets/             # Game assets (screenshots, sprites)
├── README.md           # You are here
├── LICENSE             # MIT License
└── .gitignore          # Git ignore rules
```

**Philosophy:** Single-file architecture means zero setup. Open `index.html` in any browser and play. No build step, no dependencies, no server required.

---

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure & UI panels |
| **CSS3** | Styling, animations, HUD layout |
| **Vanilla JavaScript** | All game logic, rendering, physics |
| **Canvas 2D API** | Game rendering (world, entities, character, effects) |
| **localStorage** | Save/load game progress |

**Zero external dependencies.** No frameworks, no libraries, no npm.
